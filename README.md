🚀 App Counter

Welcome to App Counter, a simple yet powerful project built to practice core JavaScript concepts and DOM manipulation.

📖 About the Project

This project is a classic counter application that dynamically updates its appearance based on the value:

🟢 Turns green when the number is greater than 0
🔴 Turns red when the number is less than 0

It’s a small project, but it focuses on logic building, which is one of the most important skills in programming.

✨ Features
➕ Increment button (increase the counter)
➖ Decrement button (decrease the counter)
🎨 Dynamic color change based on the value
⚡ Real-time DOM updates
🛠️ Technologies Used
🌐 HTML5
🎨 CSS3
⚡ JavaScript (Vanilla JS)
